# Frotas Pelo Mundo — Dossiê de Audiência

Apresentação interativa (dossiê comercial) do canal **Frotas Pelo Mundo**, a volta ao mundo em voo solo de Alexandre "Alex Bacana" Frota.

Reúne as métricas de **YouTube** e **Instagram** dos últimos 90 dias (abr–jun 2026) num deck de slides navegável, feito para apresentação a patrocinadores e parceiros.

## Ver online

Página publicada via GitHub Pages: **https://danusiojr.github.io/frotas-pelo-mundo-dossie/**

## Como usar

- Abra `index.html` no navegador (ou acesse o link do GitHub Pages).
- Navegue pelos slides com as **setas do teclado**, a **roda do mouse** ou os **pontos** na lateral.
- Para exportar em PDF: abra e use **Imprimir → Salvar como PDF** (o layout tem modo de impressão).

## Estrutura

```
index.html          → o dossiê completo (HTML autossuficiente, com Chart.js embutido)
assets/hero.mp4     → vídeo de fundo da capa
assets/reel-slide.mp4 → vídeo do slide de abertura
```

## Fontes dos dados

YouTube Studio + Instagram Insights — últimos 90 dias (abril a junho de 2026).
